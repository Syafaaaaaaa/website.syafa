Game Catur Sederhana

Website interaktif ini mengimplementasikan manipulasi *Document Object Model* (DOM) menggunakan JavaScript murni untuk merender papan catur dan mengatur pergerakan bidak secara dinamis di browser.

Developer: Ahmad Syafa Assidiq
Dosen: M. Agus Badruzaman Al Khoir, S.Kom

💡 Cara Mencoba Papan Catur:

  • Klik pada salah satu bidak (misal: Pion/Pawn di baris ke-2 atau ke-7).
  • Kotak akan berubah warna menjadi hijau setelah dipilih.
  • Klik kotak kosong tujuan untuk memindahkan bidak tersebut.
